## Concepts Covered

Over 30 days, this series touches on a wide range of real frontend skills:

**JavaScript**
- `requestAnimationFrame` and animation loops
- Canvas API and particle systems
- Linear interpolation (Lerp) and easing
- Drag API and pointer events
- TensorFlow.js and in-browser AI

**CSS**
- 3D transforms and `perspective`
- `mix-blend-mode` and filter effects
- CSS-only interactivity with `:has()` and checkboxes
- Scroll-driven animations
- Custom properties and dynamic theming

**Creative Coding**
- Physics simulation (magnetic pull, gravity, bounce)
- Procedural particle systems
- SVG path manipulation
- WebGL basics
